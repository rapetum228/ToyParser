# ToyParser
Парсинг информации с сайта https://www.toy.ru/catalog/boy_transport/
Данные для парсинга:
- Название региона;
- Хлебные крошки;
- Название товара;
- Цена;
- Цена старая;
- Раздел с наличием (В наличии товар или не в наличии);
- Ссылки на картинки;
- Ссылка на товар.

Запись ведётся в csv-файл
